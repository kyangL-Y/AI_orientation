🧹 清除浏览器缓存工具

⚠️ 为什么需要清除缓存？

菜单数据可能被缓存在浏览器的 localStorage 和缓存文件中。如果不清除，您将看到旧的菜单结构（平铺的菜单、错误的路径等）。

方法一：一键清除（推荐）

✅ 清除成功！

请刷新页面（F5）或重新登录以加载最新数据。

方法二：手动清除

如果一键清除无效，请按以下步骤操作：

  1. 按 Ctrl + Shift + Delete 打开清除浏览数据对话框
  2. 勾选以下选项：
    • ✅ Cookie 和其他网站数据
    • ✅ 缓存的图片和文件
    • ✅ 本地存储（localStorage）
  3. 时间范围选择：全部
  4. 点击"清除数据"按钮
  5. 刷新页面（F5）

方法三：强制刷新

  1. 按住 Shift 键
  2. 点击浏览器的刷新按钮
  3. 或者按 Ctrl + F5

验证是否清除成功

按 F12 打开开发者工具，切换到 Console 标签，输入：

localStorage.clear();
sessionStorage.clear();
console.log('缓存已清除');

然后刷新页面。
