## 使用说明

### 查看完整内容
1. 打开"系统管理 → 定制化配置 → 企业文化配置"
2. 富文本编辑器会显示所有内容
3. **如果内容较长，请向下滚动编辑器查看完整内容**
4. 编辑器底部会显示内容统计信息

### 编辑内容
1. 在富文本编辑器中直接编辑
2. 使用工具栏格式化文本（标题、列表、加粗等）
3. 点击"保存"按钮保存修改
4. 保存后会显示成功提示

### 内容结构建议
企业文化内容建议使用以下结构：
```html
<h2>标题1</h2>
<p>内容段落...</p>
<ul>
  <li>列表项1</li>
  <li>列表项2</li>
</ul>
<hr/>

<h2>标题2</h2>
<p>内容段落...</p>
```

- 使用 `<h2>` 标签作为章节标题
- 使用 `<p>` 标签包裹段落
- 使用 `<ul>` 和 `<li>` 创建列表
- 使用 `<hr/>` 分隔不同章节

## 用户端显示

用户端会自动解析企业文化内容：
- 按 `<h2>` 标题分割成多个卡片
- 每个卡片显示一个章节
- 使用 Bento Grid 布局美化展示
- 支持响应式设计，适配移动端

## 常见问题

### Q: 为什么看不到全部内容？
A: 内容可能超出编辑器可见区域，请向下滚动编辑器查看。编辑器支持垂直滚动，最大高度为 800px。

### Q: 如何确认内容是否完整？
A: 查看编辑器下方的内容统计信息，显示字符数和大小。完整的企业文化内容约 4500+ 字符。

### Q: 保存后用户端看不到更新？
A: 请确保：
1. 保存成功（有成功提示）
2. 用户端刷新页面（Ctrl+F5 强制刷新）
3. 检查浏览器控制台是否有错误

### Q: 如何恢复默认内容？
A: 点击"填入默认模版"按钮，会清空当前内容。然后可以重新编辑。
